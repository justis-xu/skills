# R3：AI 写作痕迹

参考：`../language/humanizer-zh.md`，按其模式逐项扫

## 重点模式（技术文章最高频）

| 模式 | 触发词 / 信号 |
|------|-------------|
| **AI 词汇** | 此外、至关重要、展示（动词滥用）、格局、值得注意的是、深入探讨、强调（动词）、充满活力、织锦 |
| **内联标题垂直列表** | 连续 2+ 个 `**词**：散文段` → 改成表格或无序列表 |
| **三段式法则** | "A、B 和 C"三项并列 → 考虑改成两项或四项 |
| **否定式排比** | "不仅…而且…"、"不仅仅是…而是…" |
| **破折号过度** | 一段内超过 1 个破折号就审查 |
| **通用积极结论** | 段尾"为…奠定了基础"、"令人期待"、"未来可期" |
| **以 -ing 结尾的肤浅分析** | "反映了…联系"、"突出了…重要性"、"彰显了…意义" |
| **俚语 / 网络用语** | 炸出来、搞定、跑通、起飞、yyds 等 → 直接替换为标准书面表达 |

## 评分

完成逐项扫描后，按 humanizer-zh 的五维标准评分：

**直接性 / 节奏 / 信任度 / 真实性 / 精炼度**，各 10 分，总分 50。

- 未发现问题 → 一句话确认"X 项 clean"
- 发现问题 → 给候选改法
- 最后给出五维评分，告知用户
